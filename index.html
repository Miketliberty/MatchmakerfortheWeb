Product Name: Airline Loadsheet

Application Overview

This aplication will allow a airline to Create and manage the load on aircraft. It will warn the user if weight goes beyond the center of gravity limits and allows the user to make changes of locations of cargo or containers

Many airlines still use paper load sheets which are often difficult to read and do not allow changes to made easily if containers need to be moved or removed from the plane.
